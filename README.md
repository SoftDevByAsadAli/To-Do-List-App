# To-Do List App

A clean, modern, and responsive To-Do List web application. Built as a beginner project for my developer portfolio. You can add tasks, mark them as complete, delete them, and clear everything. All tasks are saved in your browser using localStorage, so they stay even after you refresh the page.

## Features

- Add a new task
- Mark a task as completed (click the task text to toggle)
- Delete a single task
- Clear all tasks at once
- Tasks saved in **localStorage** (they remain after refreshing)
- Responsive design with hover effects
